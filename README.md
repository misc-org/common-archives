## common-archives ―― 部活でのルールなどの共通事項のアーカイブ

slack のメッセージ履歴の保管期限が、2022 年 9 月 1 日から、90 日に変更になるので、
部活で残しておくべきことをこのレポジトリに残します。

このレポジトリは部員以外も閲覧できます。  
不要な改ざんを防ぐため、`main`ブランチには直接書き込むことはできません。  
全て管理者の PR の承認が必要です。  
何か追加したいことや変更点があれば、気軽に PR を投げてください。
